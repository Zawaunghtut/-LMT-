# Myanmar LMT Calculator (မြန်မာမြိုပြဒေသများ၏ ဒေသိယစံတော်ချိန် တွက်ကိရိယာ)

မြန်မာနိုင်ငံရှိ မြိုပေါင်း ၂၀၀ ကျော်၏ တည်နေရာ (Latitude/Longitude) နှင့် စံတော်ချိန်ညှိကိန်း (Time Correction) များကို အခြေခံ၍ ဒေသိယစံတော်ချိန် (Local Mean Time - LMT) ကို တိကျစွာ တွက်ချက်ပေးနိုင်သော Application ဖြစ်ပါသည်။

## အင်္ဂါရပ်များ (Features)
- မြန်မာနိုင်ငံမြိုပေါင်း ၂၀၀ ကျော်၏ တည်နေရာအချက်အလက်များ ပါဝင်ခြင်း
- မြန်မာစံတော်ချိန် (MST) မှ ဒေသိယစံတော်ချိန် (LMT) သို အလိုအလျောက် တွက်ချက်ပေးခြင်း
- HTML, CSS, JavaScript တစ်ဖိုင်တည်းဖြင့် Offline အသုံးပြုနိုင်ခြင်း
- မြိုအမည်နှင့် ပြည်နယ်အမည်များဖြင့် လွယ်ကူစွာ ရှာဖွေနိုင်ခြင်း

## အသုံးပြုနည်း (Usage)
1. index.html ဖိုင်ကို browser တွင် ဖွင့်ပါ။
2. မိမိတို သိရှိလိုသော မြိုအမည်ကို ရှာဖွေ၍ ရွေးချယ်ပါ။
3. နာရီကြည့်ချိန် (MST) ကို ရိုက်ထည့်ပြီး တွက်ချက်ပါ။
